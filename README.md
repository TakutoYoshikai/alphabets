# alphabets
The alphabets printer.

### Requirements
* macOS or Ubuntu
* npm

### Usage
**install**
```bash
npm install -g TakutoYoshikai/alphabets
```

**print lowercase alphabets**
```bash
alphabets -a
```

**print uppercase alphabets**
```bash
alphabets -A
```

**print numbers**
```bash
alphabets -n
```

**print those**
```bash
alphabets
```
