# alphabets
The alphabets printer.

### Requirements
* macOS or Ubuntu
* npm

### Usage
**install**
```bash
npm install -g TakutoYoshikai/alphabets
```

**print lowercase alphabets**
```bash
alphabets
```

**print uppercase alphabets**
```bash
alphabets -A
```

**print hex characters**
```bash
alphabets -hex
# 0123456789abcdef

alphabets -HEX
# 0123456789ABCDEF
```

**print numbers**
```bash
alphabets -0
```

**print those**
```bash
alphabets -all
```
